\chapter{Repositórios e Materiais Complementares}\label{apendiceD}

Este apêndice apresenta os links para os materiais complementares desenvolvidos ao longo do trabalho, incluindo o repositório com todas as análises realizadas e o site construído para apresentar mulheres cientistas.

\section*{Repositório do Projeto}
O repositório contém todas as análises conduzidas neste trabalho, incluindo preparação dos dados, formação dos clusters, aplicação do Random Forest, cálculo dos valores SHAP, bem como os códigos utilizados para geração dos gráficos e demais procedimentos metodológicos.

\begin{itemize}
    \item \textbf{Repositório GitHub:}\\
    \url{https://github.com/SamCatarina/analises-mapeamento-perfis}
\end{itemize}

\section*{Site sobre Mulheres Cientistas}
O site desenvolvido apresenta mulheres científicas organizadas por áreas e subáreas de atuação, destacando seus principais trabalhos, contribuições e informações relevantes.

\begin{itemize}
    \item \textbf{Site Mulheres na Ciência:}\\
    \url{https://samcatarina.github.io/MulheresNaCiencia/#/}
\end{itemize}
